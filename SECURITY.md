# Security Policy

## Supported Versions

Specifically, we only provide security updates for the latest release.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in Rivery, please do not post it directly in the issues or public repositories. 

Please contact the maintainer:
- **Rheehose (Rhee Creative)**

We will aim to acknowledge your report within 48 hours and provide a fix as soon as possible.

# 보안 정책

## 지원되는 버전

현재 가장 최신 릴리즈 버전에 대해서만 보안 업데이트를 제공합니다.

| 버전 | 지원 여부          |
| ---- | ----------------- |
| 최신  | :white_check_mark: |

## 취약점 보고 방법

Rivery에서 보안 취약점을 발견한 경우, 이슈나 공용 저장소에 직접 게시하지 마십시오.

아래 제작자에게 연락해 주시기 바랍니다:
- **Rheehose (Rhee Creative)**

보고된 내용은 48시간 이내에 확인하고 가능한 한 빨리 해결책을 제공하도록 노력하겠습니다.
